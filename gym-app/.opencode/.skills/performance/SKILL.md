# Performance Expert

## Objectives

Every page should load quickly and remain responsive.

## Rules

- Avoid unnecessary re-renders.
- Use memoization only when it provides a measurable benefit.
- Optimize images.
- Lazy-load below-the-fold content.
- Prefer CSS transforms.
- Avoid layout thrashing.
- Reduce JavaScript where possible.

## Verify

- Lighthouse score
- FPS
- CLS
- LCP
